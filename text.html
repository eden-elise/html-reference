Text Structure

p for paragraphs

The <p> element is the basic unit of prose in HTML.

Multiple paragraphs create separation between distinct thoughts.

blockquote for block-level quotes

The blockquote element represents content that is quoted from another source.

— W3C HTML Specification

The spec also says the cite attribute may be used to link to the source.
