Linear Algebra Study Notes

These are personal study notes for LA based on Linear Algebra Done Right by Sheldon Axler. They cover core definitions, key theorems, and computational examples from USD coursework. Last updated March 7, 2025.

Chapter 1: Vector Spaces

1.1 Definition

A vector space is a set V over a field F together with two operations — addition and scalar multiplication — satisfying eight axioms. The field F is typically ℝ or ℂ.

The dimension of a vector space is written dimF(V). For example, ℝn has dimension n over ℝ. The zero vector space has dimension zero — a degenerate case worth remembering.

The definition of a vector space is the foundation upon which all of linear algebra is built. Understanding it deeply is more valuable than memorizing theorems.

— Linear Algebra Done Right, Sheldon Axler

As Axler notes, a list of length zero is a basis of the zero vector space — easy to forget on exams.

1.1.1 The Eight Axioms

For all u, v, w ∈ V and scalars a, b ∈ F: 

1. u + v = v + u                        (commutativity)
2. (u + v) + w = u + (v + w)            (associativity)
3. ∃ 0 ∈ V such that v + 0 = v          (additive identity)
4. ∃ −v such that v + (−v) = 0          (additive inverse)
5. 1v = v                               (multiplicative identity)
6. (ab)v = a(bv)                        (multiplicative associativity)
7. a(u + v) = au + av                   (distributivity)
8. (a + b)v = av + bv                   (distributivity)
Note on Notation

The symbol 𝔽 is used throughout Axler as a generic field standing for either ℝ or ℂ. The term scalar always refers to an element of the field, not an element of the vector space itself — this distinction matters when working over ℂ.

Historical Remark

The formal definition of a vector space was established around 1888 by Giuseppe Peano. This predates the modern axiomatic treatment by several decades.

Chapter 2: Linear Maps

2.1 Definition and Examples

A linear map is a function T : V → W satisfying additivity and homogeneity. Both conditions must hold — forgetting to check scalar multiplication over ℂ is a recurring exam mistake.

The key insight is that a linear map is completely determined by its values on a basis. Once you know where the basis vectors go, the entire map is fixed.

2.2 Corrected Definition (Quiz 3 Revision)

My original notes stated that every linear map between finite-dimensional spaces can be represented as a square matrix. This is wrong — the correct statement is a linear map T : V → W is represented by an m × n matrix where m = dim W and n = dim V, which is square only when dim V = dim W.

It is also sometimes assumed that all vector spaces are finite-dimensional — infinite-dimensional spaces like function spaces are common in analysis.

Chapter 3: Computational Notes

3.1 Python Representation

In Python, a vector in ℝ3 can be represented as a list. The expression v = [1, 2, 3] defines a simple vector. For matrix operations, import numpy as np and use np.linalg.matrix_rank(A) to check rank.

To run a cell in Jupyter, press Shift + Enter. To restart the kernel, press Ctrl + F5. A typical output after computing a row echelon form looks like:

array([[1., 0., 0.], [0., 1., 0.], [0., 0., 1.]])

this is the identity matrix: 

        | 1 0 0 |
        | 0 1 0 |
        | 0 0 1 |

This is the 3 × 3 identity matrix, confirming the three vectors are linearly independent.

Chapter 4: Terminology and Language Notes

4.1 Etymology

The term eigenvalue comes from the German Eigenwert, meaning "characteristic value." Do not confuse eigenvalues with eigenvectors — they are related but distinct objects. The equation is T(v) = λv, where λ is the eigenvalue and v is the eigenvector.

The word eigendecomposition is sometimes hyphenated on narrow screens but refers to a single concept.

In Japanese mathematics, the concept of a 固有値(eigenvalue) is written with kanji that literally translate to "characteristic value," consistent with the German origin.
