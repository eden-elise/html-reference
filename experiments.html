🔥 100 levels deep. Document what you see in DevTools.

Finding: The browser handled all 100 levels without any performance issues or visual breakage. It can clearly handle more nesting than any human would want to interact with. In DevTools, the DOM tree forms the shape of a right-pointing arrow as each level indents further right, making the structure visually obvious but enormous. In practice the real limit is not the browser at all. It is the user. Opening more than a few levels becomes genuinely maddening my vision was literally going blurry and I started thinking the arrows were moving. Any UI that required this would be unusable.

Experiment 2: Emoji as an id Attribute

Hypothesis: The HTML spec says id can be any string that does not contain spaces. That technically includes emoji. I expect the element to render fine but CSS and JavaScript targeting it will probably break or require weird escaping.

This paragraph has id="🌸"

This paragraph has id="🦋"

This paragraph has id="🌿"

Finding: I kind of thought this wasn't going to work but it actually worked all the CS and JavaScript was able to locate and use them as normal id's. The code also validates fine.

Experiment 3: <div> Inside a <p>

Hypothesis: A div inside a p is invalid because p only allows inline content. I expect the browser to either ignore the div or throw a visible error. I do not expect it to silently rewrite my HTML.

This paragraph starts normally, then a div appears:

I am a div inside a p. I should not exist here.
I am a div inside a div inside a p. This is getting weird.
I am a div inside a div inside a div inside a p. What does the DOM actually look like right now?
This text tries to continue the paragraph after the div. Is it still inside the p?

Finding: The browser rewrote the HTML without any error or warning. It auto-closed the p tag right before the div, rendered the divs as siblings, then the text that tried to continue the paragraph after the div, but it had no parent element wrapping it at all. This caused a silent structural changes that were only visible if I looked in dev tools.
